### Active model split (as of 2026-05-27)

The runtime config now distinguishes orchestrator vs sub-agent surfaces:

| Surface | Secret key | Live value | Inference-profile id |
|---|---|---|---|
| Orchestrator + LLM endpoint (high reasoning) | `llm_model_id` | Opus 4.7 | `global.anthropic.claude-opus-4-7` |
| Sub-agent (per-user worker, high volume) | `llm_subagent_model_id` | Sonnet 4.6 | `global.anthropic.claude-sonnet-4-6` |

**Important:** Bedrock requires inference-profile IDs (prefixed with `global.` / `apac.`) instead of bare model IDs for these models. Direct `InvokeModel` against `anthropic.claude-opus-4-7` returns `ValidationException: Invocation ... with on-demand throughput isn’t supported`. Always use the profile id.

Containers spawned by `src/cloud/container-manager/lifecycle.ts` get the sub-agent model id forwarded via `BEDROCK_LLM_MODEL_ID` env var; `BedrockClient` and `BedrockClaude` in `container/sub-agent/src/llm/` honour it (env > caller arg > `DEFAULT_MODEL_ID`).

## Re-running this inventory

```powershell
# Caller / region sanity
aws sts get-caller-identity
aws configure get region

# Bedrock models
aws bedrock list-foundation-models --region ap-southeast-1 `
  --query "modelSummaries[?contains(modelId,'anthropic')].{id:modelId,status:modelLifecycle.status}"

# DynamoDB
aws dynamodb list-tables --region ap-southeast-1
aws dynamodb describe-table --region ap-southeast-1 --table-name nanoclaw-chat-messages

# S3
aws s3api list-buckets --query "Buckets[?contains(Name,'nanoclaw')].Name"

# OpenSearch Serverless
aws opensearchserverless list-collections --region ap-southeast-1
aws opensearchserverless batch-get-collection --names nanoclaw-documents --region ap-southeast-1

# ElastiCache
aws elasticache describe-cache-clusters --region ap-southeast-1 --show-cache-node-info

# Secrets keys (no values)
aws secretsmanager get-secret-value --region ap-southeast-1 --secret-id nanoclaw/app-config `
  --query SecretString --output text | ConvertFrom-Json | Get-Member -MemberType NoteProperty | Select-Object -ExpandProperty Name

# ECR
aws ecr describe-images --region ap-southeast-1 --repository-name nanoclaw/orchestrator `
  --query "reverse(sort_by(imageDetails,&imagePushedAt))[:5].{tags:imageTags,pushed:imagePushedAt,size:imageSizeInBytes}"
```
