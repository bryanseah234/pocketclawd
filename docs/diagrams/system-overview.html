Clawd — System Overview

WhatsApp & Telegram AI assistant · AWS Singapore (ap-southeast-1) · account 709609992277

What it is

  • • A personal assistant in WhatsApp or Telegram
  • • No app to install — uses the chat you already have
  • • Built for busy professionals in SE Asia
  • • Powered by Claude (Anthropic) on AWS Bedrock

What it does

  • • Remembers you across conversations
  • • Reads & summarises your PDFs and links
  • • Answers from your own knowledge base
  • • Live news, weather, prices, maps
  • • Creates images, PDFs & documents
  • • Reminders & a 7am morning digest

How it runs

  • • Two AWS services that talk via a message queue
  • • Scales to many users, fully managed infrastructure
  • • All data stays in the Singapore region
  • • Credentials never leave AWS Secrets Manager
  • • Infrastructure defined as code (Terraform)
